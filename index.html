GitHub 저장소와 Google 로그인을 설정하면 팀원 모두가 사용할 수 있습니다
1
GitHub Private 저장소 생성 + PAT 발급
github.com → New repository → Private 선택 → 생성
이후 Settings → Developer settings → Personal access tokens → Fine-grained tokens → Repository contents: Read and write
2
Google OAuth 클라이언트 ID 생성
console.cloud.google.com → APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → 웹 애플리케이션
승인된 JavaScript 원본에 이 페이지의 URL 추가 (예: https://xxx.netlify.app)
3
연결 정보 입력
